Постановка задачи.
Тестовое задание разбито на модули в порядке возрастания сложности.
Таким образом мы сможем понять действительный уровень кандидата.

Задача уровень 1: Необходимо сверстать анкету обратной связи, используя JSON файл из архива.
У каждого вопроса есть свой тип и вид ответной части, пожалуйста учитывайте это при верстке.
Внешний вид и выбор контролов на ваш вкус, задание творческое.
Требование: адаптивная верстка (проверить под популярные мобильные разрешения)
Будет отлично если: используете препроцессор less или sass/scss, сборщик guld/grunt

Задача уровень 2: Необходимо реализовать валидацию заполнения анкеты. Положим, что все поля обязательны для заполнения.
Сообщения об ошибке выводить в виде popup или toast
Требование: После заполнения анкеты должна появляться кнопка "Отправить". Валидация должна включать js проверки (только css не достаточно) и запускаться по нажатию на "Отправить". В случае отсутствия ошибок должен выводится popup или toast "Спасибо за Ваше мнение" и форма должна очищаться.
Будет отлично если: валидная анкета собирается в массив объектов вида [{title:'',reference:'',value:''}, {title:'',reference:'',value:''}, ...]
где title - вопрос, reference - id вопроса, value - ответ на вопрос.
Сформированный объект должен выводится в консоль после нажатия на кнопку "Отправить".

Задача уровень 3: Необходимо реализовать кеширование процесса заполнения анкеты.
Требование: Текущий прогресс заполнения должен запоминаться на случай обновления страницы.
Если страница обновилась, данные введенные ранее должны восстановиться в блоках ответов.
Будет отлично если: выполните весь проект на одном из MVC javascript фраемворков.

Все задачи должны представлять собой единый проект. Оформлять каждую отдельно не нужно.
За каждый уровень кандидату начисляются баллы, которые будут учитываться при собеседовании.

Удачи!
